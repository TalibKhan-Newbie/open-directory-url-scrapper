# 📧 Email Scraper (URL + Google Dork)

A powerful Python-based email scraper that extracts emails from:
- 🌐 Single or multiple URLs
- 🔍 Google Dork search results

All extracted emails are saved into a CSV file automatically.

---

## 🚀 Features

- Extract emails from multiple URLs
- Google Dork based scraping
- Multi-threaded scraping (faster performance)
- Automatic CSV export
- Handles large datasets
- Unique email filtering (no duplicates)
- Works with dynamic websites using Selenium

---

## ⚙️ Requirements

- Python 3.8+
- Mozilla Firefox installed
- GeckoDriver (auto-handled or manual)
